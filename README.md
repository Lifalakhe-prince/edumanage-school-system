# EduManage School System

A full-stack Django + React project for managing students, teachers, and classes.  
Developed by ** (Lifalakhe-prince)** From Eagleview Graduate Institute as part of the EduManage project brief.

## Features
- Student registration and management
- Teacher assignment
- REST API with Django
- React frontend for easy navigation

## Setup
1. Clone the repo:  
   `git clone https://github.com/Lifalakhe-prince/edumanage-school-system.git`
2. Install dependencies
3. Run backend (Django) and frontend (React)
